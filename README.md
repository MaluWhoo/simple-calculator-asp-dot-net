# 🧮 ASP.NET Web Forms Calculator

A simple yet functional web-based calculator built with ASP.NET Web Forms. Perform basic arithmetic operations with an intuitive interface and real-time expression evaluation.

## 📸 Screenshots

![Calculator Interface](screenshot.jpeg)

## ✨ Features

- **Basic Operations**: Addition, subtraction, multiplication, division
- **Real-time Expression Display**: See your input as you build expressions
- **Error Handling**: Clear error messages for invalid expressions
- **Clean Interface**: Simple, user-friendly design
- **One-Click Clear**: Easy reset functionality

## 🚀 Technologies Used

- **ASP.NET Web Forms** - Server-side web application framework
- **C#** - Backend logic and event handling
- **.NET Framework** - Core framework
- **DataTable.Compute()** - Expression evaluation engine

## 📋 Prerequisites

- Visual Studio 2019 or later
- .NET Framework 4.5 or higher
- IIS Express or any compatible web server
